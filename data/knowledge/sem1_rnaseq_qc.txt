10 SEM1 24/25 BIS7019-B – ‘Omics’ Techniques in Biology and Healthcare
Base quality scores:
• Probability of a base call made by the sequencer being incorrect
• ASCII character → Quality (Q) score → probability of erroneous base call
ASCII_BASE 33 (most widely used quality scoring system)
Probability of error = 0.1 → 10% probability of error → base call accuracy of 90%
RNA-seq FASTQ files
11 SEM1 24/25 BIS7019-B – ‘Omics’ Techniques in Biology and Healthcare
FASTQ file integrity:
• FASTQ files often very large (> 2 GB per file in size)
• Important to check file integrity upon downloading/transferring
• Can verify integrity via a file’s ‘hash’ (checksum) – a sequence of
letters and/or numbers (theoretically) unique to that file
• Several hash generating algorithms (MD5, SHA-2, CRC32, etc)

Tools for assessing RNA-seq read quality:
• Major tool for checking RNA-seq read quality is FastQC software
• Encompasses a range of quality check metrics (‘modules’)
• Can be utilised in several ways:
Ø Through terminal/ command prompt
Ø Downloadable application with graphical user interface
Ø Using its corresponding R package (fastqcr)
Ø Using Galaxy webserver

17 SEM1 24/25 BIS7019-B – ‘Omics’ Techniques in Biology and Healthcare
Important notes when using FastQC:
• FastQC provides an indication of what’s going on
• Should interpret in context of ‘typical’ behavior for your library
• E.g., WARN/FAIL in some modules may be normal for RNA-seq
• Use flags as guides rather than definitive barriers to progress

Using FastQC – ‘fastqcr’ R package:
• Facilitates batch running of FastQC (package available via CRAN)
# Load library
library(fastqcr)
# Run fastQC on all samples in specified directory
fastqc(fq.dir = ‘path/to/FASTQ/files’,
qc.dir = ‘path/to/save/directory’,
fastqc.path = ‘path/to/fastqc/tool’)
# Combine output results of all samples
qc.agg <- qc_aggregate(qc.dir = qc.dir = ‘path/to/save/directory’)

23 SEM1 24/25 BIS7019-B – ‘Omics’ Techniques in Biology and Healthcare
Filtering and trimming:
• Can help increase number of reads aligner is able to successfully use
• Several tools exist that enable us to be able to:
1. Trim/cut/mask sequences:
a) From low quality score regions
b) Beginning/end of sequence
c) Removing adapters
2. Filter sequences:
a) With low mean quality score
b) That are too short
c) With too many unidentified (N) bases
Quality control of RNA-seq reads
